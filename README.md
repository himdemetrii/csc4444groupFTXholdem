# csc444TXholdem
A reinforcement learning agent trained to play Texas Hold 'Em poker, built with PyTorch and Google DeepMind's OpenSpiel.
This project explores applying deep reinforcement learning to the game of Texas Hold 'Em poker. The bot learns optimal play strategies through self-play and reward signals, using a custom neural network architecture built on top of OpenSpiel's game simulation framework.



Tech Stack: Python, PyTorch, Google DeepMind OpenSpiel, NumPy
